# Markdown Converter

## Summary

Architecture for a single-pass Markdown parsing library capable of incremental streaming and HTML conversion.

---

**Warning: This project is *very* much WIP! It is *not* ready for use. Do not convert Markdown from an untrusted source. Consider sanitizing the HTML output.**

---


## Tutorial

For default HTML generation you can use `MarkdownConverter.convertToHtml`:

```javascript
const html = MarkdownConverter.convertToHtml(markdown);
```

Use `nodeIterator` to iterate over block nodes while parsing Markdown incrementally:

```javascript
const myBlogEntryConverter = new TreeWalker(new CustomBlogEntryGenerator());

for (let node of MarkdownConverter.nodeIterator(markdown)) {
    // Do something with the node, for example:
    blogEntries.push(myBlogEntryConverter.convert(node));
}
```

As the returned object is an `IteratorObject`, use some functional style methods:

```javascript
const blogEntryList = MarkdownConverter.nodeIterator(markdown)
    .map(node => myBlogEntryConverter.convert(node))
    .reduce((blogEntries, entry) => blogEntries + "\n" + entry);
```

You can use `MarkdownConverter.nodeStream` to consume parsing results asynchronously as `ReadableStream`:
```javascript
async function showBlogEntries() {
   for await(let blogEntry of MarkdownConverter.nodeStream(markdown)) {
       showBlogEntry(blogEntry);
   } 
}
```

Finally, you can also analyze or change the document tree generated by `TreeBuilder`:

```javascript
const tree = MarkdownConverter.documentTree(markdown);

// ... do whatever you like with the tree before converting it to HTML

const treeWalker = new TreeWalker(new HTMLGenerator());
const html = treeWalker.convertTree(tree);
```

Feel also free to have a look at the test cases to get an impression of how the library works internally.
