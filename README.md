# Markdown Converter

## Summary

Generates HTML from Markdown. 

**Careful: This is currently a fun/educational project. It is certainly *not* ready for use.**

Consider sanitizing the HTML output, especially when converting Markdown from an untrusted source.

## Tutorial

For default HTML generation you can use the `HTMLConverter`:
```javascript
const htmlConverter = new HTMLConverter();
const html = htmlConverter.convert(markdown);
```

You can also analyze or change the tree generated by `TreeBuilder`:

```javascript
const treeBuilder = new TreeBuilder();
const tree = treeBuilder.buildDocumentTree(markdown);

// ... do whatever you like with the tree

const treeWalker = new TreeWalker(new HTMLGenerator);
const html = treeWalker.convertTree(tree);
```

Feel also free to have a look at the test cases to get an impression of how the library works internally.
